# TUTORIEL

## CREATION D'UNE BRANCHE

Pour Créer une branche, l'instruction termiunale est :
- git checkout -b nom-de-la-branche

## CREATION D'UNE PULL REQUEST

Push la branche avec la commande  :
git push -u origin feature/ma-feature

Va sur ton repository sur GitHub puis Clique sur Compare & pull request et choisis ta branche.
Ajoute :

- Titre clair
- Description détaillée
- Clique sur Create pull request
